== Description ==

Adds a Mobile Detection shortcode to use on all of your pages and post.

Wrap content in shortcode [md_is_mobile]content[/md_is_mobile] to display content only on mobile devices.


###Quick Start Guide

* [Mobile detection quick start guide at thinklandingpages.com](http://www.thinklandingpages.com/mobile-detection/)


== Installation ==


1. Upload `mobile-detection` to the `/wp-content/mobile-detection/` directory
1. Activate the car sales pages through the 'Mobile Detection' menu in WordPress
1. Click **Mobile Detection** on the admin menu to enable and set your options.


== Frequently Asked Questions ==

= Do I have to know how to program or design? =

No.  The mobile detection plugin does the programming and design.

= Is there a limit to the number of mobile detections I can make? =
No, unlimited mobile detections can be made.
